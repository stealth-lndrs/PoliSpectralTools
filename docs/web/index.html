PoliSpectralTools

Spectral collocation tools for boundary-value problems and PDEs in Julia.

julia> using Pkg; Pkg.add(url="https://github.com/<user>/PoliSpectralTools.jl")
julia> using PoliSpectralTools

Core Capabilities

Collocation

Lobatto grids (Chebyshev & Legendre) with scaled differentiation matrices via Collocation.build_grid.

BVP Solvers

Linear/nonlinear solvers enforcing Dirichlet, Neumann, and Robin data with optional analytic derivatives.

PDE Stack

Diffusion (MOL + RK4), wave (leapfrog), and Poisson 2D (Kronecker) solvers.

Diagnostics

scripts/run_full_report.jl summarizes test sets and coverage in console + markdown form.
